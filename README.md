# 📊 Spectra Studio Streaming Platform Dashboard (Power BI)

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** built for **Spectra Studio**, a fictional global streaming platform.  
The dashboard analyzes streaming content using the **Spectra Titles dataset** and provides insights into content distribution, genre popularity, and platform growth trends.

The project demonstrates **data visualization, dashboard design, and business intelligence skills using Power BI**.

---

## 🛠 Tools & Technologies
- Power BI
- Data Visualization
- CSV Dataset
- Business Intelligence
- Data Analysis

---

## 📂 Dataset Information

The dashboard uses the **Spectra Titles dataset**, which contains details about movies and TV shows available on the platform.

### Dataset Fields
- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed In)
- Description

Total records analyzed: **9,687 titles**

---

# 📊 Dashboard Features

## 1️⃣ KPI Cards
The dashboard includes three key metrics:

- **Total Titles**
- **Total Genres**
- **Total Directors**

These provide a quick overview of the platform’s content library.

---

## 2️⃣ Total Titles by Country (Map)
A **map visualization** shows the geographical distribution of titles.

Insights include:
- Number of titles produced in each country
- Global content distribution
- Bubble size representing the number of titles

---

## 3️⃣ Titles by Year and Type (Line Chart)

A **line chart** displays the number of titles released over time.

Features:
- X-axis: Release Year
- Y-axis: Number of Titles
- Categories: Movies vs TV Shows

This helps analyze **content growth trends**.

---

## 4️⃣ Top 5 Genres (Bar Chart)

A **bar chart** highlights the most popular genres on the platform.

Examples include:
- Drama
- Comedy
- Suspense
- Animation
- Kids content

---

## 5️⃣ Titles by Type (Donut Chart)

A **donut chart** shows the percentage distribution of:

- Movies
- TV Shows

Example distribution:
- Movies: **80.82%**
- TV Shows: **19.18%**

---

## 6️⃣ Quarter-Based Slicer

An interactive **quarter slicer** allows filtering the dashboard by when content was added.

Options include:
- Select All
- Quarter 1
- Quarter 2
- Quarter 3
- Quarter 4

Selecting a quarter dynamically updates all dashboard visuals.
